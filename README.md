# weatherINFO_Visualize
r 을 사용한 미세먼지, 온습도 시각화 템플릿

다음 기준에 따라 지도 및 다양한 그래프로 시각화하게 됩니다.
1. 기간전체
2. 일별
3. 시간별
4. 서울 전역
5. 자치구별
6. 법정동별 


사용된 데이터의 측정기간 -> 2020.05.06 ~ 2020.05.09 , 2021.05.06 ~  2021.05.09
